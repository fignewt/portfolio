# Newton Grady - Portfolio

A professional portfolio website featuring a serene pond background with animated koi fish, showcasing my skills in computer engineering, embedded systems, and enterprise tools.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Light/Dark Theme** - Toggle between light and dark modes with persistent preference
- **Animated Koi Fish** - Four uniquely animated koi swimming in different paths
- **Lily Pads** - Interactive lily pad backgrounds for content sections
- **Terminal Boot Sequence** - Retro-themed startup animation
- **Ambient Music** - Optional background audio control
- **Professional Navigation** - Smooth scrolling and active link highlighting

## Sections

- **Home** - Hero section with name and introduction
- **About** - Professional background and experience
- **Skills** - Programming languages, development tools, and enterprise platforms

## Technologies

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (no frameworks)
- Font Awesome Icons
- Google Fonts

## Setup

Simply open `index.html` in a web browser or deploy to a web server.

## Files

- `index.html` - Main portfolio file (single-file architecture)
- `resume.pdf` - Professional resume
- `README.md` - This file
